As a consequence of linear equations of motion

For a fixed-fixed string of length \(l\), the solution of the 1D wave equation \(\partial^2\xi/\partial x^2 - c_0^{-2}{\partial^2\xi}/{\partial t^2} = 0\) is \[\xi(x,t) = \sin(n\pi x/l)[A \sin (2\pi t/\tau_n) + B\cos (2\pi t/\tau_n)]\,,\] where \(n = 1, 2, 3, \dots\), and where \(\tau_n = 2l/cn\). Consider the \(n = 1\) mode, for which \begin{equation}\label{eq:string:tau} \tau_n = 2l/c\,. \end{equation} In Ref. [1], Beyer extends the relation from adiabatic invariance \begin{align}\label{eq:string:adiabat} \frac{F\,dl}{\langle E\rangle} = \frac{d\tau}{\tau} \end{align} to discuss the radiation force in this situation. Combining Eqs. \eqref{eq:string:tau} and \eqref{eq:string:adiabat} yields the radiation force, \begin{align}\label{eq:string:adiabat:1} \boxed{F = \frac{d\tau}{dl} \frac{\langle E\rangle}{\tau} = \frac{2}{c} \frac{\langle E\rangle}{2l/c} = {\langle E\rangle}/{l}\,.} \end{align}
