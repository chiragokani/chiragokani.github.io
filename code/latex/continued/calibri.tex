\section*{Using \LaTeX\space to set Calibri and {\fontspec{Cambria}Cambria Math}}
\LaTeX\space was used to make this document appear as if it were created using Microsoft Word. It uses Calibri for text and {\fontspec{Cambria}Cambria Math} for math. For example, the acoustic wave equation is
\begin{align}
\nabla^2 p - \frac{1}{c_0^2}\frac{\partial^2p}{\partial t^2} = 0\,,
\end{align}
while two of Maxwell's equations are
\begin{align}
  \nablav \cdot \vec{E} = \frac{\rho}{\varepsilon_0}, \qquad
  \nablav \times \vec{B} =
  \mu_0 \vec{J} + \mu_0 \varepsilon_0 \frac{\partial \vec{E}}{\partial t}
\end{align}
Here's the continuity equation:
\begin{align}
\frac{\partial \rho}{\partial t} + \nablav \cdot (\rho \vec{u}) = 0.
\end{align}
Having come come full circle stylistically, I fall silent.\\
